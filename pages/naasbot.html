NaasBot – Custom Automation Framework

Relevant Experience: Dec 2020 – Sept 2022

NaasBot is a Python-based automation framework designed from scratch to orchestrate network services, server provisioning, DNS/Firewall management, and IT service operations. It delivered reusable modules, simplified integrations, and reduced operational effort across multiple accounts.

Key Deliverables & Services

  • Validation API (1.0, 2.0, 3.0)
  • Cronicle setup and R&D scheduling application
  • Static host lifecycle management (create/modify/delete)
  • DNS alias automation
  • Firewall rule automation
  • Change and incident management automation
  • Physical server decommission workflows
  • Ongoing production support for delivered services
